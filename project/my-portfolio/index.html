About Project

It is a static site that is generated through Python script. Python script reads data stored in json format and generates appropriate HTML from it. The script also uses the watchdog library to view <em>src</em> director's files for any changes, whenever a file is modified it copies all assets to the <em>build</em> directory and creates new HTML files.
